# Interactive Cosmic UI

Interactive Cosmic UI is an immersive frontend portfolio built using HTML, CSS, and JavaScript. 
It features canvas-based starfield rendering, orbital motion logic, parallax effects, 
and event-driven animations to create a cinematic space-themed user experience.

##  Tech Stack

- HTML5
- CSS3 (Animations, Transforms, Parallax Effects)
- JavaScript (Canvas API, requestAnimationFrame, DOM Manipulation)
